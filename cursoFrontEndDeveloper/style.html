Funciones de las propiedades de CSS más utilizadas

  • width: define el ancho de un elemento
  • height: define el alto de un elemento
  • background: define el color o la url de fondo de un elemento
  • background-color: define el color de fondo de un elemento
  • color: define el color de fondo de un elemento; los colores se pueden por nombres, sist. hexadecimal o por RGB
  • border: define el tamaño, estilo y color del borde de un elemento
  • border-radius: define que tan redondeado quiero mi elemento
  • margin: define el margen de un elemento. margin-right, margin-left, margin-top, margin-bottom para especificar si solo quiero un lado; también colocando en orden, de acuerdo a las manecillas del reloj, comenzando desde top (arriba) separados por un espacio
  • padding: define la distancia del borde de un elemento hasta su contenido. Si quiero especificar un padding un solo lado o solo algunos también se utiliza el método del margin
  • font-size: define el tamaño de la fuente
  • font-family: define la familia de tipografía de la fuente
  • opacity: determina la transparencia del elemento, tiene valores entre 0 y 1
  • outline: es el esquema de los elementos. ESQUEMA es una línea que se dibuja alrededor de los elementos que hace que se destaquen, generalmente de color azul si no queremos esa linea escribimos outline: none; tambien se puede modificar, ej.: outline-style: solid; outline-color: red; outline-width: 5px;
  • box-sizing: cuando queremos que un padding no afecte al tamaño total del elemento
  • transition: para cambiar los valores de las transiciones. Una transición NO es una animación. Una TRANSICION va de un punto A a un punto B sin interrupciones o saltos en medio
  • animation: permite que animemos nuestros elementos

Display en CSS

  • Block: Flujo de bloque de escritura es de arriba hacia abajo
  • Inline: Todos los elementos de este display se visualiza de uno al lado de otro de izquierda a derecha
  • Inline-block: este es similar al Inline solo que respeta el width y el height que se le asigna a los elementos
  • None: este display se utiliza para ocultar elementos sin eliminarlos por completo del DOM
  • Table: los elementos con este display se comportan como los elementos table
  • Flex: por defecto alineará los elementos hijos de izquierda a derecha, suele ir acompañado por otras propiedades como justify-content y align-items
  • Grid: al igual que flexbox, para implementarlo debemos hacer uso de display: grid en el elemento contenedor o elemento padre

Posicionamiento en CSS

  • Position: relative; el elemento se coloca en relación a su posición normal
  • Position: absolute; Se ubican en relación al elemento relativo más cercano. Aquí podemos hacer uso de propiedades top, bottom, left y right
  • Position: fixed; Se ubica en relación con la ventana del navegador y también se ve afectado por las propiedades top, bottom, left y right
  • Position: sticky; el elemento se posiciona en función de la posición de desplazamiento del usuario. Si lo ubicas a la izquierda, siempre va a estar ahí sin importar cuántas veces se haga scroll en la página, por ejemplo
  • Position: static; los elementos en HTML son estáticos por defecto, siguen el flujo normal de la página y no se ven afectados por las propiedades top, bottom, left y right
  • Position: initial; Ubica el elemento en la posición que tiene de forma predeterminada
  • Position: inherital; hereda esta propiedad de su elemento padre
